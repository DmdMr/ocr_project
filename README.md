# Система распознавания рукописных документов

Веб-приложение для распознавания рукописного текста с изображений,
анализа и хранения данных.

## Локальный запуск на Windows

1. Установите:
   - Python 3.10+
   - Node.js LTS (с npm)
   - MongoDB (опционально как Windows Service `MongoDB`)
2. Из корня проекта запустите `start-local.bat`.
3. Скрипт:
   - поднимет `.venv` (если нет),
   - установит backend-зависимости,
   - запустит backend на `http://localhost:8000`,
   - установит frontend-зависимости,
   - запустит frontend на `http://localhost:5173`.

Чтобы остановить процессы, используйте `stop-local.bat`.
